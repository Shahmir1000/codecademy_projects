Website Styleing

  1. Color Pallet

    • coolors

      This page has a section for trending color palettes. If you want a safe option of colours I recommend checking this page out.

    • mycolor

      This link is cool because you can give a colour you are working with and it will Generate different color palettes with it. You can give it any type of value.

  2. Gradient

    • uigradients

      This page has a list of premade gradients. To check them out click Show all gradients under the logo on the top left hand side of the screen. Once you have clicked on a Gradient you like, press enter to access the css for it.

    • cssgradient

      You can make a custom gradient from this page adding multiple colors to your design. Your previous gradient is saved and it give you the css at the botom of the page.
